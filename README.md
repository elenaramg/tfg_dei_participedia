# tfg_dei_participedia

Análisis automatizado de diversidad, equidad e inclusión (DEI) en casos de participación ciudadana documentados en [Participedia](https://participedia.net).

## Descripción

Este proyecto desarrolla un pipeline de análisis computacional que combina expresiones regulares y un modelo de lenguaje (GPT-4o mini) para detectar y clasificar grupos sociales y sus roles en procesos de participación ciudadana.
